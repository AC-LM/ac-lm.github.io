PyQt5(1)——安装

@LM | 2020-12-01

PyQt5是对Qt所有类的Python封装，结合Python简洁的语法用户可以使用PyQt5高效、简单地开发出自己想要的程序。

pip install PyQt5
pip install PyQt5-tools
# 安装工具，QTdesigner路径Lib\site-packages\qt5_applications\Qt\bin\designer.exe
