Python(23)——os模块文件操作

@LM | 2021-03-25

参考文档： os文档 @Python文档

os.listdir()

os.listdir() 方法用于返回指定的文件夹包含的文件或文件夹的名字的列表。它不包括隐藏文件如（. 或 .. 开头的文件）

os.listdir(path)
# path -- 需要列出的目录路径
# 返回指定路径下的文件和文件夹列表

import os, sys

path = "/var/www/html/"
dirs = os.listdir(path)

for file in dirs:
    print(file)

os.path.exists()

os.path.exists() 方法用于判断文件夹是否存在

os.path.exists(path)
# path -- 需要列出的目录路径

import os
path = "/var/www/html/"
if os.path.exists(path):
    pass

os.mkdir()

os.mkdir() 方法用于创建文件夹

import os

path="/var/www/html/ABC"
isExists=os.path.exists(path)

if not isExists:
    os.mkdir(path)
    print('创建成功')
else:
    print('目录已存在')
