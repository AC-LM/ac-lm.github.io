Windows—利用搜索实现批量文件移动

Jan 13, 2021

1.实现

  1. 找到目标文件夹
  2. 查看文件夹中文件的格式，如.jpg .png .txt
  3. 在确定需要移动的文件格式后，在右上角搜索框，输入格式如.txt 正则匹配文件名
  4. 在出现的搜索结果中，按住Ctrl+A选择全部，按住Ctrl+C复制全部，然后，在其他地方新建文件夹，按住Ctrl+V复制所有选择的文件到新文件中即可
