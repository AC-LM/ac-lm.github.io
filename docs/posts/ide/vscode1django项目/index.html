VSCode(1)——Django项目

@LM | 2021-08-26

环境配置

IDE 运行环境需有 Python 与 Django

项目创建

  1. 在终端以命令 django-admin startproject XXXX 创建项目
  2. 在 VSCode 的扩展管理栏 Extensions 下载 Python 与 Django 插件
  3. 进入调试，添加 Python > Django
  4. 完成配置，启动项目
