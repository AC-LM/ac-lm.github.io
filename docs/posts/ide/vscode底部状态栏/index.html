VSCode—底部状态栏

Aug 27, 2021

1.改变颜色

  1. 在设置中搜索workbench.colorCustomizations
  2. 点击编辑setting.json进行修改
{
    "workbench.colorCustomizations": {
        "statusBar.background" : "#008cff",
        "statusBar.noFolderBackground" : "#008cff",
        "statusBar.debuggingBackground": "#008cff",
    },
}
