Nov 15, 2021

lscpu 中 socket、core、thread 的意义 @Whosemario的家

1.lscpu

Linux 上使用 lscpu 可以查看CPU的信息

> lscpu

2.Socket

Socket 就是主板上插 CPU 的槽的数量

3.Core

Core 就是平时说的核，双核、四核等，就是每个 CPU 上的核数

4.Thread

Thread 就是每个 Core 上的硬件线程数，即超线程

对操作系统来说，其逻辑 CPU 的数量就是 SocketCoreThread
