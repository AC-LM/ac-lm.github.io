Python(8)—字典取值

Dec 10, 2020

1.取键：keys()

bb={'人才/可怕':23,'伏地魔&波特':'army','哈哈哈,人才,回合':'hhh'}
for ii in bb.keys():
    print(ii)
#--输出-------------------------------------------------------------------------
人才/可怕
伏地魔&波特
哈哈哈,人才,回合

2.取值：values()

for jj in bb.values():
    print(jj)
#--输出-------------------------------------------------------------------------
23
army
hhh

4.取 键.值 对：items()

for kk,vv in bb.items():
    print(kk, vv)
#---输出------------------------------------------------------------------------
人才/可怕 23
伏地魔&波特 army
哈哈哈,人才,回合 hhh

5.获取指定键的值

c = bb.get('人才','没有找到该键')
print(c)
cc = bb.get('人才/可怕','没有找到该键')
print(cc)
#----输出------------------------------------------------------------------------
没有找到该键
23

6.获取值，并且pop出这一键值对

d = bb.pop('人才','pop失败')  
print(d)
dd = bb.pop('人才/可怕','pop失败')  
print(dd)
print(bb) 
#----输出-----------------------------------------------------------------------
pop失败
23
！！字典变化了！！
